# 🎀 Agrinho Premium: O Campo com Carinho

Um site interativo, educativo e visualmente encantador desenvolvido para a competição **Agrinho**. O projeto aborda temas essenciais como **Agroforte**, **Reforma Agrária (MST)** e **Futuro Sustentável**, utilizando um design estilo **Coquette** (feminino, fofo, com lacinhos e tons pastéis) para engajar o público de forma leve e informativa.

![Tecnologia](https://img.shields.io/badge/Tech-HTML5%20%7C%20CSS3%20%7C%20JS-pink)

## 📸 Demonstração

O site conta com uma experiência visual rica, incluindo chuva de pétalas animada, abas interativas e quizzes dinâmicos.

## ✨ Funcionalidades (Features)

-   **🌸 Chuva de Pétalas (Canvas):** Uma animação suave de pétalas caindo no fundo da tela usando JavaScript e HTML5 Canvas.
-   **🎀 Design Coquette:** Interface visual moderna com paleta de cores rosa pastel, menta e creme, utilizando fontes elegantes e ícones de lacinhos.
-   **📚 Conteúdo Rico e Interativo:**
    -   **Agroforte:** Seção com estatísticas animadas e acordeões (accordions) para detalhar tecnologias agrícolas.
    -   **MST (Reforma Agrária):** Conteúdo organizado em **Abas (Tabs)** para facilitar a leitura sobre Luta, Produção e Educação.
    -   **Futuro Sustentável:** **Checklist interativo** onde o usuário pode marcar hábitos sustentáveis e ver sua pontuação subir em tempo real.
-   **🧠 Quiz Educativo:** Um jogo de perguntas e respostas para testar o conhecimento adquirido, com feedback visual e placar final.
-   **📱 Responsividade:** O site se adapta perfeitamente a celulares, tablets e computadores.
-   **🚀 Animações de Scroll:** Os elementos aparecem suavemente (fade-in) conforme o usuário rola a página.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando apenas tecnologias nativas da web (sem frameworks pesados), garantindo leveza e facilidade de edição.

-   **HTML5:** Estrutura semântica e acessível.
-   **CSS3:** Estilização avançada, Flexbox, Grid, animações `@keyframes` e variáveis CSS.
-   **JavaScript (ES6+):** Lógica de interatividade, manipulação do DOM, API de Canvas e Observers para animações de scroll.
-   **Fontes:** Google Fonts (`Playfair Display`, `Dancing Script`, `Nunito`).
-   **Ícones:** FontAwesome 6.

## 🚀 Como Instalar e Rodar

Siga estes passos simples para executar o projeto no seu computador:

1.  **Crie uma pasta:** Crie uma nova pasta no seu computador chamada `projeto-agrinho`.
2.  **Crie os arquivos:** Dentro dessa pasta, crie 3 arquivos de texto e renomeie-os para:
    -   `index.html`
    -   `style.css`
    -   `script.js`
3.  **Copie o código:** Abra cada arquivo e cole o código correspondente fornecido anteriormente.
    -   Cole o código HTML no `index.html`.
    -   Cole o código CSS no `style.css`.
    -   Cole o código JavaScript no `script.js`.
4.  **Abra o site:** Dê um clique duplo no arquivo `index.html`. Ele abrirá no seu navegador (Chrome, Edge, Firefox, etc).

> **Nota:** É necessário estar conectado à internet para carregar as fontes e ícones, e para exibir as imagens de exemplo.

## 📖 Temas Abordados

1.  **Agroforte:** Foca na modernização do campo, uso de drones, irrigação inteligente e aumento da produtividade com tecnologia.
2.  **MST (Reforma Agrária):** Aborda a luta por justiça social, a importância da terra para quem trabalha nela, a agroecologia (produção sem veneno) e a educação no campo.
3.  **Futuro Sustentável:** Enfatiza a reciclagem, economia de água, energia limpa e o consumo consciente.
